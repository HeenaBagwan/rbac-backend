# RBAC Backend


1. Copy `.env.example` to `.env` and set values
2. `npm install`
3. `npm run seed` to seed permissions, roles and admin user
4. `npm run dev` to start server


API endpoints:
- /api/auth/register
- /api/auth/login
- /api/roles
- /api/users
